# Cafe24 빌더 공용 보드 CSS 백업 가이드

이 디렉터리는 Cafe24 웹빌더 공용 가상 경로 (`/bizdemo92513/inc/board/`) 하위의 보드 전용 스타일시트(CSS)들을 독립 서버로 이관할 때 백업하여 보관하는 경로입니다.

## 백업 및 이관 절차
Cafe24 이외의 단독 웹 호스팅(AWS, 가비아 등)으로 이전할 경우, 외부 보안 정책으로 인해 가상 경로(`/bizdemo92513/`) 아래의 리소스들이 로드되지 않아 게시판이 깨지게 됩니다. 다음 절차를 통해 리소스를 독립화할 수 있습니다.

1. **에셋 다운로드:**
   기존 Cafe24 호스팅이 구동되고 있는 상태에서 브라우저 개발자 도구(F12)의 Network 탭 또는 주소창을 통해 아래의 스타일시트 내용을 복사하여 이 디렉터리에 동일한 파일명으로 저장합니다.
   - `board_1.css`
   - `board_3.css`
   - `board_4.css`

2. **경로 변경 (리팩토링):**
   다운로드 완료 후, 공통 헤더 혹은 개별 게시판 파일 내에서 아래와 같이 경로를 수정합니다.
   - **기존:** `/bizdemo92513/inc/board/board_X.css`
   - **변경:** `<?=$default_path?>/inc/board/board_X.css`
